# Gas Market Dashboard (Excel)

This project is an Excel based dashboard designed to visualise short term developments in the European natural gas market. It consolidates key market fundamentals and price data into one interactive interface, making it easier to analyse daily movements, structural changes and fundamental drivers.

## Features
TTF price tracking  
• Day ahead and within day TTF price levels  
• Daily price changes and trend indicators  

Fundamental inputs  
• European storage levels  
• LNG import volumes  
• Pipeline flows  
• Outages and maintenance events  
• Temperature deviations  

Automatic calculations  
• Daily percentage and absolute price changes  
• 7 day rolling average  
• Basic volatility metric  

Interactive dashboard  
• Date selector that updates all KPIs  
• Clean layout with clear charts and tables  

## Purpose
The dashboard was created to practise market analysis and understand how fundamental changes affect short term gas prices.

## Tech stack
Excel only  
No macros required  
Works on both Windows and Mac
